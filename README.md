# UNAME - Username Generator

UNAME is a **fun and interactive username generator** built with HTML, CSS, and JavaScript. It allows you to create usernames in different styles with options for numbers, symbols, and underscores, and control the username length using a slider. Perfect for gamers, developers, anime fans, or anyone looking for a unique username!

---

## Features

- **Multiple Styles:** Anime, Gamer, Dev, Aesthetic, Dark, Random
- **Custom Options:** Add numbers, symbols, and underscores
- **Dynamic Length:** Adjust username length with a slider
- **Copy to Clipboard:** Quickly copy generated usernames
- **Favorite Icon:** Mark favorite usernames
- **Responsive & Interactive:** Loader animation during generation and hover effects for buttons

---

## How to Use

1. Open `index.html` in your browser.
2. Select a **style** by clicking one of the buttons (Anime, Gamer, Dev, Aesthetic, Dark, Random).
3. Choose **options** like numbers, symbols, or underscores.
4. Adjust the **count slider** for username length.
5. Click **GENERATE** to get unique usernames.
6. Click the **Copy** button to copy a username to your clipboard.
7. Click the **heart icon** to mark your favorite usernames.
